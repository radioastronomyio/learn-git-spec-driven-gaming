# Staging

Pre-commit staging area for work in progress. Files here are not yet ready for their final location in the repository. Agents and humans use this directory to draft, iterate, and validate content before moving it to its permanent home.

---

## 5. Conventions

Purpose: Staging is a workspace, not a permanent home. Files should move to their target directory once reviewed and approved. If a file has been in staging for an extended period, it is either stalled work or belongs in `recycle/`.

No structure requirements: Files in staging can use any naming or organization that makes sense for the work in progress. Structure is imposed when files move to their final location.
